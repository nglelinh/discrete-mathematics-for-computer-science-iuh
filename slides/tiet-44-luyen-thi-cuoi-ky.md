# Chiều sâu & Liên hệ (Tổng hợp)

**Các chủ đề trọng tâm cần nắm sâu**:
- Chứng minh + Định lý.
- Mô hình hóa logic + đếm.
- Tối ưu Boolean.
- Phân tích truy hồi + đồ thị.

**Liên hệ xuyên suốt**:
Tất cả chủ đề đều dùng "đếm + chứng minh + mô hình hóa rời rạc".

**Chiến lược thi**:
- Luôn viết định nghĩa trước khi chứng minh.
- Dùng ví dụ nhỏ để kiểm tra.
- Kết nối các phần (ví dụ: DNF → Karnaugh → mạch).
