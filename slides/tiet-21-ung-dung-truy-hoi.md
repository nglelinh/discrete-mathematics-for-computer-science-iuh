# Ứng dụng Hệ thức Truy hồi

**Toán Rời Rạc — IUH**

Tiết 21 (45 phút)

---

# Mục tiêu

- Áp dụng hệ thức truy hồi vào thuật toán
- Phân tích độ phức tạp đệ quy
- Ví dụ thực tế: Fibonacci, Hanoi, Merge Sort

---

# Nội dung

- Tháp Hà Nội: T(n) = 2T(n-1) + 1 → O(2^n)
- Merge Sort: T(n) = 2T(n/2) + n → O(n log n)
- Fibonacci và các dãy số

**Phân tích đúng đắn**:
- Sử dụng quy nạp để chứng minh công thức
- Big-O từ nghiệm hệ thức

---

**Liên hệ**:

- Truy hồi ↔ Đồ thị.
