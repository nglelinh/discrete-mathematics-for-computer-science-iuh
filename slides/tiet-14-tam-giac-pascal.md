# Công thức Nhị thức

\[(x + y)^n = \sum_{k=0}^{n} \binom{n}{k} x^{n-k} y^k\]

**Chứng minh bằng tổ hợp**:
Số cách chọn k vật từ n loại trong (x+y) mở rộng.

**Chứng minh bằng quy nạp**:
Base + giả thiết + bước quy nạp.

---

# Thực hành

Mở rộng (a + b)^5, (2x - 3y)^4.

---

# Chiều sâu & Liên hệ

**Chứng minh**:
- Bằng tổ hợp: hệ số chọn k.
- Bằng quy nạp: base + giả thiết.

**Liên hệ**:
- Nhị thức ↔ Đếm.
- Nhị thức ↔ Xác suất (phân phối nhị thức).

**Ứng dụng nâng cao**:
- Mở rộng nhị thức trong phân tích thuật toán.
- Sinh hệ số trong mã hóa.
