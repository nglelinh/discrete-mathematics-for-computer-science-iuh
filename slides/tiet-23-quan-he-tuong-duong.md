# Quan hệ Tương đương + Lớp Tương đương

**Toán Rời Rạc — IUH**

Tiết 23 (45 phút)

---

# Mục tiêu

- Định nghĩa quan hệ tương đương
- Lớp tương đương và phân hoạch
- Ứng dụng trong CSDL và phân loại

---

# Định nghĩa

**Quan hệ tương đương** trên tập A:
- Phản xạ
- Đối xứng
- Bắc cầu

**Lớp tương đương** của a: [a] = {x | x R a}

**Định lý**: Các lớp tương đương tạo thành một phân hoạch của A.

---

# Ứng dụng

- Phân loại sinh viên theo GPA
- Modular arithmetic (số dư)
- CSDL: nhóm bản ghi

---

# Chiều sâu & Liên hệ

**Định lý**: Các lớp tương đương tạo thành một phân hoạch (partition) của tập.

**Liên hệ**:
- Tương đương ↔ Đồ thị (các thành phần liên thông).
- Tương đương ↔ Đếm (số lớp = chỉ số).

**Ứng dụng nâng cao**:
- Modular arithmetic (mã hóa).
- Phân loại dữ liệu trong CSDL.
- Type equivalence trong compiler.
