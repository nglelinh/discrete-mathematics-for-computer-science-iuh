# Các tiên đề

1. Giao hoán: a + b = b + a , a · b = b · a
2. Kết hợp
3. Phân phối
4. Phần tử đơn vị: a + 0 = a , a · 1 = a
5. Bổ sung: a + a' = 1 , a · a' = 0
